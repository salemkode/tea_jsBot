# bot telegram (عبود)

هذة النسخه هي نسخة جديده من البوت السابقي المبني ب لغة جافا بينما هذة مبنية بلغة جافاسكربت 

### Hot to download && run this bot

1- download this repo

```
$ git clone https://github.com/superastorh/nodeBot.git
```

2- install this bot

```
$ cd nodeBot
$ npm install
```

3- Run this bot

```
$ npm start
```
