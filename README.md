# bot telegram (عبود)

### Hot to download && run this bot

1- download this repo

```bash
$ git clone https://github.com/superastorh/tea_jsBot.git
```

2- install this bot

```
$ cd tea_jsBot
$ npm install
```

3- Run this bot

```
$ npm start
```
